## Supermarket-mvc
### A simple Java Swing application demonstrating the MVC (Model-View-Controller) architecture for managing a supermarket's customers, items, and orders.

### Features

- Customer Management: Add, update, delete, and search customers
- Item Management: Add, update, delete, and search inventory items
- Order Processing: Create orders with multiple items, apply discounts, and automatically update inventory

### Technologies Used

- Java Swing for GUI
- JDBC for database connectivity
- MySQL for database
- NetBeans IDE for development

### Key Functionality
Customer Form

- Save new customers
- Update existing customer details
- Delete customers
- View all customers in a table

Item Form

- Save new items with quantity on hand (QoH)
- Update item information
- Delete items
- View all items in a table

Order Form

- Search and select customers
- Search and select items
- Add multiple items to cart
- Apply discounts per item
- Place orders (updates inventory automatically)
- Transaction management for data consistency
  
This project was developed as a self-study project to learn and understand the MVC architecture pattern.
